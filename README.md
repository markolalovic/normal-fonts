# normal-fonts

This repository contains the code to draw the letters of the English alphabet as the contours of the density function of a mixture of bivariate normal distributions.

Here is the notebook where you can play with the fonts:

[https://observablehq.com/@markolalovic/fun-with-statistics-normal-fonts](https://observablehq.com/@markolalovic/fun-with-statistics-normal-fonts)

I picked up this idea for the initials logo from the blog post here:

[http://www.ikosmidis.com/post/blog-logo/](http://www.ikosmidis.com/post/blog-logo/)

and just generalized it to all the letters of the English alphabet.
